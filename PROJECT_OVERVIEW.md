# 项目说明

## 仓库定位

本仓库用于整理本科阶段 C++ 学习项目，重点体现面向对象编程、数据结构算法实现和小型实践项目。

## 内容分类

- `cpp/Account inherit`：账户继承练习，包含基类和派生类设计。
- `cpp/Account inherit virtual`：虚继承相关练习。
- `cpp/ch10_重载运算符`：运算符重载章节练习。
- `cpp/Clion/CL_Algorithm`：排序、查找、高精度阶乘等算法实现。
- `cpp/Clion/CL_Datastruct`：顺序表、链表、栈、树、KMP、AVL、哈夫曼树等数据结构实现。
- `cpp/Clion/CL_problem`：算法题和问题求解练习。
- 其他目录：课程实验、小型控制台项目和 Visual Studio 练习项目。

## 学习重点

- 类与对象
- 继承、多态、虚继承
- 运算符重载
- C++ 数据结构实现
- CMake / Visual Studio / CLion 项目组织

## 运行说明

包含 Visual Studio 和 CMake/CLion 两类项目：

- `.sln` / `.vcxproj` 项目可使用 Visual Studio 打开。
- `CMakeLists.txt` 项目可使用 CLion 或命令行 CMake 构建。

## 整理说明

仓库已过滤 IDE 缓存和编译产物，例如 `.vs`、`cmake-build-*`、`Debug`、`Release`、`*.exe` 等。下载后需要重新编译。
